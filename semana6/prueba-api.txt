Se realizó una petición POST a la ruta:

http://localhost:3000/registro

Enviando el siguiente JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor respondió correctamente con:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto demuestra que Express recibió los datos mediante req.body y los devolvió en formato JSON.
