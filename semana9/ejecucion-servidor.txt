Al ejecutar el comando node server.js, Node.js inicia el servidor Express y comienza a escuchar solicitudes en el puerto 3000.

En la terminal aparece el mensaje:
"Servidor ejecutándose en puerto 3000".

Cuando se visita la ruta /estado en el navegador, el servidor responde con un objeto JSON que indica que el servidor está funcionando y que el servicio corresponde a una API comunitaria.
