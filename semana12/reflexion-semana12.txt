Durante este curso aprendí que la tecnología puede ser una herramienta importante para fortalecer la participación ciudadana y el trabajo comunitario. Comprendí cómo un sistema puede integrar bases de datos, automatización y generación de documentos para resolver necesidades reales. También aprendí la importancia de proteger los datos personales y utilizar únicamente la información necesaria. El uso de herramientas como Git y GitHub facilita el desarrollo y la organización de proyectos tecnológicos. Además, entendí cómo las automatizaciones permiten mejorar la comunicación con los usuarios. La tecnología no solo sirve para desarrollar aplicaciones, sino también para promover procesos más transparentes y eficientes. Finalmente, comprendí que las soluciones digitales pueden acercar a las comunidades y contribuir al desarrollo social cuando son utilizadas de manera responsable.
